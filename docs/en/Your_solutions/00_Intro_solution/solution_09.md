## 9. Optimization Problem — Solution

The curve is defined as:

$$y = 3 - x^2$$

A rectangle in the first quadrant has its base on the x-axis,
with one corner at the origin and the opposite corner on the curve at point $(x,\ y)$.

---

### Step 1: Express the Area

$$A(x) = x \cdot y = x(3 - x^2)$$

$$A(x) = 3x - x^3$$

---

### Step 2: Find the First Derivative

$$A'(x) = 3 - 3x^2$$

---

### Step 3: Set $A'(x) = 0$ to Find Critical Points

$$3 - 3x^2 = 0$$

$$x^2 = 1$$

$$\boxed{x = 1} \quad (\text{since } x > 0 \text{ in the first quadrant})$$

---

### Step 4: Second Derivative Test

$$A''(x) = -6x$$

At $x = 1$:

$$A''(1) = -6 < 0$$

Since $A''(1) < 0$, this confirms a **maximum**.

---

### Step 5: Find the Dimensions

$$x = 1$$

$$y = 3 - (1)^2 = 3 - 1 = 2$$

---

### Step 6: Maximum Area

$$A = x \cdot y = 1 \times 2 = \boxed{2 \text{ square units}}$$

---

### Result

| Dimension | Value |
|---|---|
| Width | $x = 1$ |
| Height | $y = 2$ |
| Maximum Area | $A = 2$ |

The rectangle with maximum area has dimensions $\boxed{1 \times 2}$.
