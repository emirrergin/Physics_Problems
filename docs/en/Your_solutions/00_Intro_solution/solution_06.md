## 6. Function Analysis — Solution

Given function:

$$f(x) = 3x^2 - 12x + 7$$

---

### Step 1: Find the First Derivative

$$f'(x) = 6x - 12$$

---

### Step 2: Set $f'(x) = 0$ to Find Critical Points

$$6x - 12 = 0$$

$$6x = 12$$

$$\boxed{x = 2}$$

---

### Step 3: Second Derivative Test

$$f''(x) = 6$$

At $x = 2$:

$$f''(2) = 6 > 0$$

Since $f''(2) > 0$, the function is **concave up** at $x = 2$, which confirms a **local minimum**.

---

### Step 4: Find the Value of the Minimum

$$f(2) = 3(2)^2 - 12(2) + 7$$

$$= 12 - 24 + 7$$

$$\boxed{f(2) = -5}$$

---

### Result

| | Value |
|---|---|
| Critical point | $x = 2$ |
| Type | Local **minimum** |
| Minimum value | $f(2) = -5$ |

There is **no local maximum** since $f''(x) = 6 > 0$ for all $x$, meaning the parabola opens **upward** everywhere.
